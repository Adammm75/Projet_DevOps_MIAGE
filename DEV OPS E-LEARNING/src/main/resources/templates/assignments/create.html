Conseils pour créer un devoir
Titre
  • Utilisez une numérotation claire (TP1, TP2...)
  • Indiquez le sujet principal
Classes
  • Sélectionnez les classes concernées par ce devoir
  • Seuls les étudiants des classes sélectionnées verront le devoir
Description
  • Définissez clairement les objectifs d'apprentissage
  • Listez les livrables attendus
  • Précisez les critères d'évaluation
  • Indiquez les ressources recommandées
Date limite
  • Donnez suffisamment de temps aux étudiants
  • Évitez les périodes d'examens
  • Communiquez la date en avance
Notation
  • Utilisez un barème cohérent
  • Préparez une grille d'évaluation
  • Prévoyez des critères objectifs
Important

Une fois le devoir créé, les étudiants inscrits à ce cours recevront une notification. Vous pourrez modifier les détails tant qu'aucune soumission n'a été faite.
